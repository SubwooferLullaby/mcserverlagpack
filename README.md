# McServerLagPack - by Clumber
This repository is dedicated to a detailed analysis of minecraft java server mechanics and what causes lag and or delay. This repository is intended to be used for educational purposes to further help develop plugins / research.

# Features:
* Tutorials on how to replicate the machines. 
* Detailed instructions on the inner workings of lag machines
* Schematics
* Various methods of lag such as entity or packet.
